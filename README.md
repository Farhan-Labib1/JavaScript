## Projektbeschreibung

In diesem Projekt habe ich 3 Spiele mit einer Suchfunktion Programmiert. 1. Spiel: **Zahlenraten Spiel**, 2. Spiel: **Schere, Stein, Papier** und das letzte Spiel: **Münzenwurf Spiel**. Man kann mit der Suchfunktion
entscheiden welches Spiel man gerne spielen möchte.

## Verwendung von 3 Spiel in einem Programm

<ol>
  <li>Man muss eingeben, welches Spiel man spielen möchte. Es wird als Hinweis stehen, was man drücken muss.</li>
  <li>Dann kann man schon spielen.</li>
  <li>Es wird als Hinweis stehen, was man drücken muss, wenn man weiter spielen oder zurück zu Suchfunktion gehen möchte, sowie auch noch für die Programm Beendung.</li>
</ol>
